Lesson IX

Essential Java Classes

Explore wrapper classes and essential utility classes that enhance Java programming capabilities.

Lesson Overview

Java provides essential classes that wrap primitive types into objects and offer utility methods. Understanding these classes is crucial for working with collections, parsing, and object-oriented programming.

Real-World Context: Wrapper classes are used when you need to store primitives in collections (ArrayList), work with null values, or use utility methods for parsing and conversion. They bridge the gap between primitive types and object-oriented Java.

Learning Outcomes

  • Understand the purpose of wrapper classes
  • Convert between primitive types and wrapper objects
  • Use wrapper class utility methods for parsing and conversion
  • Apply autoboxing and unboxing concepts

Autoboxing and Unboxing

Java 
public class AutoboxingDemo {
    public static void main(String[] args) {
        // Autoboxing: primitive to wrapper (automatic)
        Integer num1 = 100;  // int -> Integer
        Double num2 = 3.14;   // double -> Double
        
        // Unboxing: wrapper to primitive (automatic)
        int value1 = num1;    // Integer -> int
        double value2 = num2; // Double -> double
        
        // Works in expressions too
        Integer a = 10;
        Integer b = 20;
        int sum = a + b;  // Auto-unboxed for arithmetic
        
        System.out.println("Sum: " + sum);
    }
}

2. Wrapper Class Utility Methods

Parsing Strings to Numbers

Java 
public class ParsingDemo {
    public static void main(String[] args) {
        // Parse String to primitive
        int num1 = Integer.parseInt("42");
        double num2 = Double.parseDouble("3.14159");
        boolean flag = Boolean.parseBoolean("true");
        
        // Parse String to wrapper object
        Integer obj1 = Integer.valueOf("100");
        Double obj2 = Double.valueOf("2.718");
        
        // Convert number to String
        String str1 = Integer.toString(42);
        String str2 = String.valueOf(3.14);
        
        System.out.println("Parsed int: " + num1);
        System.out.println("Parsed double: " + num2);
    }
}

Useful Constants and Methods

Java 
public class WrapperMethods {
    public static void main(String[] args) {
        // Integer constants
        System.out.println("Max int: " + Integer.MAX_VALUE);
        System.out.println("Min int: " + Integer.MIN_VALUE);
        
        // Comparison methods
        int result = Integer.compare(10, 20);  // -1 (10 < 20)
        int max = Integer.max(15, 25);       // 25
        int min = Integer.min(15, 25);       // 15
        
        // Character methods
        System.out.println(Character.isDigit('5'));     // true
        System.out.println(Character.isLetter('A'));    // true
        System.out.println(Character.toUpperCase('a')); // A
        
        // Number base conversion
        System.out.println(Integer.toBinaryString(10)); // 1010
        System.out.println(Integer.toHexString(255));   // ff
    }
}

Summary

  • Wrapper classes provide object representations of primitive types.
  • Autoboxing automatically converts primitives to wrappers; unboxing does the reverse.
  • Use parseInt(), parseDouble() etc. to convert Strings to primitives.
  • Wrapper classes provide useful constants like MAX_VALUE and MIN_VALUE.
  • Character class offers methods for character classification and conversion.
