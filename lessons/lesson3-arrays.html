Lesson III

Arrays

Learn to store and manipulate collections of data efficiently using one-dimensional and multi-dimensional arrays.

← Previous: Iterative Structures
Next: Functions →

Lesson Overview

Arrays are fundamental data structures that allow you to store multiple values of the same type in a single variable. Instead of declaring separate variables for each value, arrays let you organize related data in a structured, indexed collection.

Real-World Context: Arrays are everywhere: a student's grades throughout the semester, daily temperatures for a month, pixels in an image, or inventory items in a warehouse system. Any time you need to work with a collection of similar items, arrays are the solution.

Learning Outcomes

At the end of this lesson, the student should be able to:

  • Illustrate the memory allocation and indexing of one-dimensional and multi-dimensional arrays.
  • Analyze different array traversal techniques to locate or modify specific data elements.
  • Design and Implement a data-management program that organizes and summarizes multi-dimensional datasets.

1. Introduction to Arrays

An array is a container object that holds a fixed number of values of a single type. The length of an array is established when the array is created and cannot be changed after creation.

Key Characteristics

  • Fixed size: Once created, the array size cannot change
  • Homogeneous: All elements must be of the same data type
  • Zero-indexed: The first element is at index 0
  • Contiguous memory: Elements are stored in consecutive memory locations
Why Use Arrays?
  • Efficient storage of multiple related values
  • Easy access using index numbers
  • Simplifies code when working with collections
  • Enables use of loops for batch processing

2. One-Dimensional Arrays

A one-dimensional array is a linear list of elements, like a row of boxes where each box can hold one value.

Declaration Syntax

Java Syntax 
// Declaration only
dataType[] arrayName;

// Declaration with size
dataType[] arrayName = new dataType[size];

// Declaration with initialization
dataType[] arrayName = {value1, value2, value3};

Example: Creating Arrays

Java 
public class ArrayBasics {
    public static void main(String[] args) {
        // Method 1: Declare and allocate separately
        int[] numbers;
        numbers = new int[5];  // Creates array of 5 integers (default: 0)
        
        // Method 2: Declare and allocate together
        double[] prices = new double[3];
        
        // Method 3: Declare and initialize with values
        String[] days = {"Mon", "Tue", "Wed", "Thu", "Fri"};
        
        // Accessing elements
        System.out.println("First day: " + days[0]);  // Mon
        System.out.println("Third day: " + days[2]);  // Wed
        
        // Modifying elements
        numbers[0] = 10;
        numbers[1] = 20;
        numbers[2] = 30;
        
        // Array length property
        System.out.println("Days array length: " + days.length);  // 5
    }
}

Example: Student Grades

Java 
public class StudentGrades {
    public static void main(String[] args) {
        // Array of quiz scores
        int[] quizScores = {85, 92, 78, 95, 88};
        
        // Calculate total and average
        int total = 0;
        for (int i = 0; i < quizScores.length; i++) {
            total += quizScores[i];
        }
        
        double average = (double) total / quizScores.length;
        
        System.out.println("Total score: " + total);
        System.out.printf("Average score: %.2f%n", average);
    }
}
ArrayIndexOutOfBoundsException: Accessing an index outside the valid range (0 to length-1) throws this runtime exception. Always ensure index values are within bounds.

3. Multi-Dimensional Arrays

Multi-dimensional arrays are arrays of arrays. The most common is the two-dimensional (2D) array, which can be visualized as a table with rows and columns.

2D Array Syntax

Java Syntax 
// Declaration and allocation
dataType[][] arrayName = new dataType[rows][columns];

// Declaration with initialization
dataType[][] arrayName = {
    {row0col0, row0col1, row0col2},
    {row1col0, row1col1, row1col2},
    {row2col0, row2col1, row2col2}
};

Example: Grade Book

Java 
public class GradeBook {
    public static void main(String[] args) {
        // 2D array: 3 students, 4 subjects
        int[][] grades = {
            {85, 90, 78, 92},  // Student 1
            {88, 76, 95, 89},  // Student 2
            {92, 88, 84, 96// Student 3
        };
        
        String[] subjects = {"Math", "Science", "English", "History"};
        
        // Access specific element: Student 2, Subject 3 (English)
        System.out.println("Student 2 English grade: " + grades[1][2]);
        
        // Print all grades in table format
        System.out.println("\n         Math  Sci   Eng   Hist");
        System.out.println("--------------------------------");
        
        for (int student = 0; student < grades.length; student++) {
            System.out.print("Student " + (student + 1) + ": ");
            for (int subject = 0; subject < grades[student].length; subject++) {
                System.out.printf("%5d ", grades[student][subject]);
            }
            System.out.println();
        }
    }
}

Example: Matrix Operations

Java 
public class MatrixAddition {
    public static void main(String[] args) {
        // Two 2x3 matrices
        int[][] matrixA = {
            {1, 2, 3},
            {4, 5, 6}
        };
        
        int[][] matrixB = {
            {7, 8, 9},
            {10, 11, 12}
        };
        
        // Result matrix
        int[][] result = new int[2][3];
        
        // Add matrices
        for (int i = 0; i < 2; i++) {
            for (int j = 0; j < 3; j++) {
                result[i][j] = matrixA[i][j] + matrixB[i][j];
            }
        }
        
        // Print result
        System.out.println("Matrix A + Matrix B:");
        for (int i = 0; i < 2; i++) {
            for (int j = 0; j < 3; j++) {
                System.out.print(result[i][j] + " ");
            }
            System.out.println();
        }
    }
}

4. Array Traversal

Array traversal means visiting each element in an array. There are several ways to traverse arrays in Java.

Method 1: Traditional For Loop

Java 
int[] numbers = {10, 20, 30, 40, 50};

// Forward traversal
for (int i = 0; i < numbers.length; i++) {
    System.out.println("Element at index " + i + ": " + numbers[i]);
}

// Backward traversal
for (int i = numbers.length - 1; i >= 0; i--) {
    System.out.println("Element at index " + i + ": " + numbers[i]);
}

Method 2: Enhanced For Loop (For-Each)

Java 
int[] numbers = {10, 20, 30, 40, 50};

// Enhanced for loop - simpler syntax when index not needed
for (int num : numbers) {
    System.out.println("Value: " + num);
}

Method 3: While Loop

Java 
int[] numbers = {10, 20, 30, 40, 50};
int i = 0;

while (i < numbers.length) {
    System.out.println("Element: " + numbers[i]);
    i++;
}

Common Array Operations

Java 
public class ArrayOperations {
    public static void main(String[] args) {
        int[] numbers = {45, 12, 78, 34, 56, 89, 23};
        
        // Find maximum value
        int max = numbers[0];
        for (int num : numbers) {
            if (num > max) {
                max = num;
            }
        }
        System.out.println("Maximum: " + max);
        
        // Find minimum value
        int min = numbers[0];
        for (int num : numbers) {
            if (num < min) {
                min = num;
            }
        }
        System.out.println("Minimum: " + min);
        
        // Calculate sum
        int sum = 0;
        for (int num : numbers) {
            sum += num;
        }
        System.out.println("Sum: " + sum);
        
        // Linear search
        int target = 34;
        int index = -1;
        for (int i = 0; i < numbers.length; i++) {
            if (numbers[i] == target) {
                index = i;
                break;
            }
        }
        System.out.println(target + " found at index: " + index);
    }
}

Common Mistakes and Best Practices

Common Mistakes

  • Off-by-one errors: Remember arrays are zero-indexed. An array of length 5 has valid indices 0-4, not 1-5.
  • ArrayIndexOutOfBoundsException: Accessing arr[arr.length] is always wrong; the last valid index is arr.length - 1.
  • Not initializing arrays: Array elements have default values (0 for numbers, null for objects), but forgetting to set values leads to bugs.
  • Confusing length vs size: For arrays, use .length (property), not .length() (method for Strings).
  • Using == for array comparison: This compares references, not contents. Use Arrays.equals() instead.

Best Practices

  • Use meaningful names: studentGrades is better than arr or data.
  • Use enhanced for loop when possible: It's cleaner and less error-prone when you don't need the index.
  • Check bounds before accessing: Validate indices before using them, especially with user input.
  • Use Arrays utility class: Arrays.toString(), Arrays.sort(), and Arrays.copyOf() are helpful methods.
  • Consider ArrayList for dynamic sizing: If you need to add/remove elements, use ArrayList instead of arrays.

Knowledge Check

1. What is the index of the first element in a Java array?

2. Given int[] arr = new int[5];, what is the default value of arr[2]?

3. How do you get the length of an array named data?

4. In a 2D array int[][] matrix = new int[3][4];, how many total elements can it store?

Summary

  • Arrays store multiple values of the same type in contiguous memory locations.
  • Arrays are zero-indexed; the first element is at index 0, the last at length - 1.
  • Array size is fixed after creation and accessed via the length property.
  • Multi-dimensional arrays are arrays of arrays, useful for tables and matrices.
  • Use traditional for loops when you need the index; use enhanced for loops for simpler iteration.
  • Common operations include finding min/max, calculating sums, and searching.
  • Always validate indices to avoid ArrayIndexOutOfBoundsException.
