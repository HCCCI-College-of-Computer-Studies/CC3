Lesson VII

Accessing Files

Learn to read from and write to files, implementing CRUD operations and processing CSV data.

Lesson Overview

File handling allows programs to persist data beyond the program's execution. This lesson covers reading, writing, updating, and deleting file data, including working with CSV files commonly used for data exchange.

Real-World Context: Applications use files to store configurations, user preferences, logs, reports, and data exports. CSV files are widely used for spreadsheet data, database exports, and data exchange between systems.

Learning Outcomes

  • Read data from text files using various Java I/O classes
  • Write and append data to files
  • Implement CRUD (Create, Read, Update, Delete) operations on file data
  • Parse and generate CSV files

1. File Handling C.R.U.D. Operations

Creating and Writing to Files

Java 
import java.io.*;

public class FileCreate {
    public static void main(String[] args) {
        // Using FileWriter (overwrites existing file)
        try (FileWriter writer = new FileWriter("output.txt")) {
            writer.write("Hello, World!\n");
            writer.write("This is line 2.\n");
            writer.write("File created successfully!");
            System.out.println("File written successfully.");
        } catch (IOException e) {
            System.out.println("Error: " + e.getMessage());
        }
        
        // Using PrintWriter for formatted output
        try (PrintWriter pw = new PrintWriter("data.txt")) {
            pw.println("Student Records");
            pw.printf("Name: %s, Score: %d%n", "Juan", 95);
            pw.printf("Name: %s, Score: %d%n", "Maria", 88);
        } catch (FileNotFoundException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

Reading from Files

Java 
import java.io.*;
import java.util.Scanner;

public class FileRead {
    public static void main(String[] args) {
        // Method 1: Using Scanner
        try (Scanner scanner = new Scanner(new File("data.txt"))) {
            while (scanner.hasNextLine()) {
                String line = scanner.nextLine();
                System.out.println(line);
            }
        } catch (FileNotFoundException e) {
            System.out.println("File not found!");
        }
        
        // Method 2: Using BufferedReader
        try (BufferedReader br = new BufferedReader(
                new FileReader("data.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            System.out.println("Error reading file!");
        }
    }
}

Appending to Files

Java 
import java.io.*;

public class FileAppend {
    public static void main(String[] args) {
        // true = append mode (doesn't overwrite)
        try (FileWriter fw = new FileWriter("log.txt", true);
             PrintWriter pw = new PrintWriter(fw)) {
            
            pw.println("New log entry: " + java.time.LocalDateTime.now());
            System.out.println("Entry appended to log.");
            
        } catch (IOException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

2. Accessing CSV Files

CSV (Comma-Separated Values) files store tabular data as plain text, with values separated by commas and rows separated by newlines.

Reading CSV Files

Java 
import java.io.*;

public class ReadCSV {
    public static void main(String[] args) {
        String filename = "students.csv";
        
        try (BufferedReader br = new BufferedReader(
                new FileReader(filename))) {
            
            String line;
            boolean isHeader = true;
            
            while ((line = br.readLine()) != null) {
                // Split line by comma
                String[] values = line.split(",");
                
                if (isHeader) {
                    System.out.println("=== Header ===");
                    isHeader = false;
                }
                
                // Process each column
                for (String value : values) {
                    System.out.print(value.trim() + "\t");
                }
                System.out.println();
            }
        } catch (IOException e) {
            System.out.println("Error reading CSV: " + e.getMessage());
        }
    }
}

Writing CSV Files

Java 
import java.io.*;

public class WriteCSV {
    public static void main(String[] args) {
        String[][] students = {
            {"ID", "Name", "Course", "GPA"},
            {"001", "Juan Dela Cruz", "BSIT", "3.5"},
            {"002", "Maria Santos", "BSCS", "3.8"},
            {"003", "Pedro Garcia", "BSIT", "3.2"try (PrintWriter pw = new PrintWriter("students.csv")) {
            for (String[] row : students) {
                pw.println(String.join(",", row));
            }
            System.out.println("CSV file created successfully!");
        } catch (FileNotFoundException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

Knowledge Check

1. What does CRUD stand for?

2. Which parameter in FileWriter enables append mode?

Summary

  • Use FileWriter and PrintWriter to write text to files.
  • Use BufferedReader or Scanner to read files line by line.
  • Pass true as second argument to FileWriter for append mode.
  • CSV files use commas to separate values; use split(",") to parse.
  • Always use try-with-resources to ensure files are properly closed.
