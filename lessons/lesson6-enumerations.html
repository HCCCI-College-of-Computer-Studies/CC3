Lesson VI

Enumerations (Enums)

Learn to define your own data types with a fixed set of named constants for type-safe programming.

Lesson Overview

An enumeration (enum) is a special data type that allows you to define a collection of named constants. Enums make your code more readable, maintainable, and less prone to errors by restricting variables to a predefined set of values.

Real-World Context: Enums are perfect for representing fixed categories: days of the week, months, compass directions, playing card suits, order statuses (pending, shipped, delivered), user roles (admin, user, guest), or traffic light colors.

Learning Outcomes

  • Define and declare enumeration types in Java
  • Use enums in switch statements and comparisons
  • Access enum values and iterate over them
  • Add methods and constructors to enums

1. Defining Enums

Basic Syntax

Java 
// Define an enum for days of the week
enum Day {
    MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY
}

// Define an enum for traffic light colors
enum TrafficLight {
    RED, YELLOW, GREEN
}

public class EnumBasics {
    public static void main(String[] args) {
        // Declare and initialize enum variables
        Day today = Day.WEDNESDAY;
        TrafficLight light = TrafficLight.GREEN;
        
        System.out.println("Today is: " + today);
        System.out.println("Light is: " + light);
        
        // Compare enums
        if (today == Day.WEDNESDAY) {
            System.out.println("It's midweek!");
        }
    }
}

Using Enums in Switch Statements

Java 
enum Season {
    SPRING, SUMMER, FALL, WINTER
}

public class SeasonExample {
    public static void main(String[] args) {
        Season current = Season.SUMMER;
        
        switch (current) {
            case SPRING:
                System.out.println("Flowers are blooming!");
                break;
            case SUMMER:
                System.out.println("Time for the beach!");
                break;
            case FALL:
                System.out.println("Leaves are falling!");
                break;
            case WINTER:
                System.out.println("Bundle up, it's cold!");
                break;
        }
    }
}

2. Enum Methods and Properties

Java 
enum Priority {
    LOW, MEDIUM, HIGH, CRITICAL
}

public class EnumMethods {
    public static void main(String[] args) {
        // values() - returns array of all enum constants
        System.out.println("All priorities:");
        for (Priority p : Priority.values()) {
            System.out.println("  " + p);
        }
        
        // ordinal() - returns position (0-based index)
        Priority task = Priority.HIGH;
        System.out.println("\nHIGH ordinal: " + task.ordinal());  // 2
        
        // name() - returns the constant name as String
        System.out.println("Name: " + task.name());  // HIGH
        
        // valueOf() - convert String to enum
        Priority fromString = Priority.valueOf("CRITICAL");
        System.out.println("From string: " + fromString);
    }
}

Try It Yourself

Explore enum methods like values(), ordinal(), name(), and valueOf(). Try iterating through all values!

3. Enums with Constructors and Fields

Java 
enum Planet {
    // Constants with values
    MERCURY(3.303e+23, 2.4397e6),
    VENUS(4.869e+24, 6.0518e6),
    EARTH(5.976e+24, 6.37814e6),
    MARS(6.421e+23, 3.3972e6);
    
    // Fields
    private final double mass;    // in kilograms
    private final double radius;  // in meters
    
    // Constructor
    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }
    
    // Getter methods
    public double getMass() { return mass; }
    public double getRadius() { return radius; }
    
    // Calculate surface gravity
    public double surfaceGravity() {
        double G = 6.67300E-11;
        return G * mass / (radius * radius);
    }
}

public class PlanetDemo {
    public static void main(String[] args) {
        for (Planet p : Planet.values()) {
            System.out.printf("%s: gravity = %.2f m/s²%n", 
                p, p.surfaceGravity());
        }
    }
}

Knowledge Check

1. What is the ordinal value of the first constant in an enum?

2. Which method returns all constants of an enum?

Summary

  • Enums define a fixed set of named constants, improving type safety.
  • Use values() to get all constants, ordinal() for position, name() for String representation.
  • Enums work well with switch statements.
  • Enums can have fields, constructors, and methods like regular classes.
