Course Overview

This course focuses on Object-Oriented Programming (OOP) using Java. You will learn how to design, develop, and implement software solutions using OOP principles. By the end of this course, you will be able to create complete applications with graphical user interfaces, file handling, and proper error management.

What You Will Learn

  • Object-Oriented Programming fundamentals - Understanding classes, objects, and how they interact
  • Java programming language syntax and structure - Variables, data types, operators, and control flow
  • Classes, objects, inheritance, and polymorphism - The four pillars of OOP
  • Exception handling and file I/O - Managing errors and working with files
  • Graphical User Interface (GUI) development - Building interactive desktop applications

Introduction to Java

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It was developed by James Gosling at Sun Microsystems (now owned by Oracle) and released in 1995.

Key Features of Java:

  • Platform Independent: Java code is compiled into bytecode that runs on any system with a Java Virtual Machine (JVM). This is known as "Write Once, Run Anywhere" (WORA).
  • Object-Oriented: Everything in Java is an object, making code modular, flexible, and easy to maintain.
  • Simple and Familiar: Java syntax is similar to C/C++ but removes complex features like pointers and multiple inheritance.
  • Secure: Java provides a secure environment with no explicit pointers and bytecode verification.
  • Robust: Strong memory management and exception handling make Java programs reliable.
  • Multithreaded: Java supports concurrent execution of multiple threads.

Your First Java Program - Explained Line by Line

Let's break down a simple "Hello World" program to understand Java's basic structure:

// This is a single-line comment - used to explain code

/*
 * This is a multi-line comment
 * Used for longer explanations
 */

/**
 * This is a Javadoc comment - used for documentation
 * @author YourName
 */
public class HelloWorld {
    // 'public' - accessible from anywhere
    // 'class' - keyword to define a class
    // 'HelloWorld' - name of the class (must match filename)
    
    public static void main(String[] args) {
        // 'public' - accessible from anywhere
        // 'static' - belongs to the class, not an instance
        // 'void' - returns nothing
        // 'main' - entry point of the program
        // 'String[] args' - command line arguments
        
        System.out.println("Hello, World!");
        // 'System' - a built-in class
        // 'out' - output stream connected to console
        // 'println' - method that prints and adds new line
        
        System.out.print("This doesn't add a new line");
        System.out.println(" - but this does!");
    }
}

Understanding the Structure:

  • Class Declaration: Every Java program must have at least one class. The class name must match the filename (HelloWorld.java).
  • Main Method: The main method is the entry point where program execution begins. Every standalone Java application must have a main method.
  • Statements: Each statement ends with a semicolon (;). Statements are executed sequentially from top to bottom.
  • Curly Braces: Used to define the beginning and end of classes, methods, and code blocks.

Java Data Types

Java is a strongly-typed language, meaning every variable must have a declared type. There are two categories of data types:

Primitive Data Types (8 types):

// Integer types
byte smallNumber = 127;           // 8-bit, range: -128 to 127
short mediumNumber = 32000;       // 16-bit, range: -32,768 to 32,767
int normalNumber = 2000000000;    // 32-bit, most commonly used
long bigNumber = 9000000000L;     // 64-bit, note the 'L' suffix

// Floating-point types
float decimal1 = 3.14f;           // 32-bit, note the 'f' suffix
double decimal2 = 3.14159265359;  // 64-bit, more precise, commonly used

// Character type
char letter = 'A';                // 16-bit Unicode character
char unicode = '\u0041';          // Unicode representation of 'A'

// Boolean type
boolean isJavaFun = true;         // Can only be true or false
boolean isHard = false;

Reference Data Types:

// String - sequence of characters (not a primitive!)
String greeting = "Hello, Java!";
String name = "John Doe";

// Arrays - collection of same-type elements
int[] numbers = {1, 2, 3, 4, 5};
String[] names = {"Alice", "Bob", "Charlie"};

// Objects - instances of classes
Scanner input = new Scanner(System.in);
Student student = new Student();

Variables and Constants

Variables store data that can change during program execution. Constants store values that never change.

public class VariablesDemo {
    public static void main(String[] args) {
        // Variable declaration and initialization
        int age;              // Declaration - reserves memory
        age = 25;             // Initialization - assigns a value
        
        int score = 100;      // Declaration and initialization together
        
        // Multiple variables of same type
        int x = 10, y = 20, z = 30;
        
        // Constants - use 'final' keyword (UPPERCASE naming convention)
        final double PI = 3.14159;
        final int MAX_STUDENTS = 40;
        final String SCHOOL_NAME = "ABC University";
        
        // PI = 3.14;  // ERROR! Cannot change a constant
        
        // Variable naming rules:
        // - Must start with letter, underscore (_), or dollar sign ($)
        // - Cannot start with a number
        // - Cannot use reserved keywords (int, class, public, etc.)
        // - Case-sensitive (age, Age, AGE are different variables)
        
        // Naming conventions (camelCase for variables):
        int studentAge = 20;
        String firstName = "John";
        double accountBalance = 1500.50;
    }
}

Basic Input and Output

Java uses the Scanner class to read input from the user and System.out for output.

import java.util.Scanner;  // Must import Scanner class

public class InputOutputDemo {
    public static void main(String[] args) {
        // Create Scanner object to read input
        Scanner scanner = new Scanner(System.in);
        
        // Reading different types of input
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();  // Reads entire line
        
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();       // Reads an integer
        
        System.out.print("Enter your GPA: ");
        double gpa = scanner.nextDouble(); // Reads a decimal number
        
        // Clear the buffer after nextInt/nextDouble
        scanner.nextLine();
        
        System.out.print("Enter your address: ");
        String address = scanner.nextLine();
        
        // Output with formatting
        System.out.println("=== Student Information ===");
        System.out.println("Name: " + name);
        System.out.println("Age: " + age + " years old");
        
        // Formatted output using printf
        System.out.printf("GPA: %.2f%n", gpa);  // 2 decimal places
        System.out.printf("%-20s %s%n", "Address:", address);
        
        // Format specifiers:
        // %s - String
        // %d - Integer
        // %f - Floating-point
        // %.2f - Float with 2 decimal places
        // %n - New line (platform independent)
        // %-20s - Left-aligned, 20 characters wide
        
        scanner.close();  // Always close the scanner
    }
}

Operators in Java

Operators are symbols that perform operations on variables and values.

public class OperatorsDemo {
    public static void main(String[] args) {
        // Arithmetic Operators
        int a = 10, b = 3;
        System.out.println("a + b = " + (a + b));   // Addition: 13
        System.out.println("a - b = " + (a - b));   // Subtraction: 7
        System.out.println("a * b = " + (a * b));   // Multiplication: 30
        System.out.println("a / b = " + (a / b));   // Division: 3 (integer)
        System.out.println("a % b = " + (a % b));   // Modulus (remainder): 1
        
        // Increment and Decrement
        int x = 5;
        System.out.println("x++ = " + x++);  // Post-increment: prints 5, then x becomes 6
        System.out.println("++x = " + ++x);  // Pre-increment: x becomes 7, prints 7
        System.out.println("x-- = " + x--);  // Post-decrement: prints 7, then x becomes 6
        System.out.println("--x = " + --x);  // Pre-decrement: x becomes 5, prints 5
        
        // Comparison Operators (return boolean)
        int p = 10, q = 20;
        System.out.println("p == q: " + (p == q));  // Equal to: false
        System.out.println("p != q: " + (p != q));  // Not equal: true
        System.out.println("p > q: " + (p > q));    // Greater than: false
        System.out.println("p < q: " + (p < q));    // Less than: true
        System.out.println("p >= q: " + (p >= q));  // Greater or equal: false
        System.out.println("p <= q: " + (p <= q));  // Less or equal: true
        
        // Logical Operators
        boolean t = true, f = false;
        System.out.println("t && f: " + (t && f));  // AND: false
        System.out.println("t || f: " + (t || f));  // OR: true
        System.out.println("!t: " + !t);            // NOT: false
        
        // Assignment Operators
        int num = 10;
        num += 5;   // num = num + 5;  -> 15
        num -= 3;   // num = num - 3;  -> 12
        num *= 2;   // num = num * 2;  -> 24
        num /= 4;   // num = num / 4;  -> 6
        num %= 4;   // num = num % 4;  -> 2
    }
}

Control Flow Statements

Control flow statements allow you to make decisions and repeat actions in your programs.

import java.util.Scanner;

public class ControlFlowDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // IF-ELSE Statement
        System.out.print("Enter your score: ");
        int score = scanner.nextInt();
        
        if (score >= 90) {
            System.out.println("Grade: A - Excellent!");
        } else if (score >= 80) {
            System.out.println("Grade: B - Very Good!");
        } else if (score >= 70) {
            System.out.println("Grade: C - Good");
        } else if (score >= 60) {
            System.out.println("Grade: D - Passed");
        } else {
            System.out.println("Grade: F - Failed");
        }
        
        // SWITCH Statement - better for multiple specific values
        System.out.print("Enter day number (1-7): ");
        int day = scanner.nextInt();
        
        switch (day) {
            case 1:
                System.out.println("Monday");
                break;  // Important! Prevents fall-through
            case 2:
                System.out.println("Tuesday");
                break;
            case 3:
                System.out.println("Wednesday");
                break;
            case 4:
                System.out.println("Thursday");
                break;
            case 5:
                System.out.println("Friday");
                break;
            case 6:
            case 7:
                System.out.println("Weekend!");
                break;
            default:
                System.out.println("Invalid day");
        }
        
        // FOR Loop - when you know how many times to repeat
        System.out.println("\nCounting 1 to 5:");
        for (int i = 1; i <= 5; i++) {
            System.out.println("Count: " + i);
        }
        
        // WHILE Loop - when condition controls repetition
        System.out.println("\nCountdown:");
        int countdown = 5;
        while (countdown > 0) {
            System.out.println(countdown);
            countdown--;
        }
        System.out.println("Blast off!");
        
        // DO-WHILE Loop - executes at least once
        int number;
        do {
            System.out.print("Enter a positive number: ");
            number = scanner.nextInt();
        } while (number <= 0);
        System.out.println("You entered: " + number);
        
        scanner.close();
    }
}

Setting Up Your Environment

To write and run Java programs, you need to set up your development environment:

Step 1: Install JDK (Java Development Kit)

  • Download JDK from Oracle's website or use OpenJDK
  • Run the installer and follow the instructions
  • Set the JAVA_HOME environment variable to your JDK installation path
  • Add the JDK's bin folder to your system PATH

Step 2: Verify Installation

// Open Command Prompt/Terminal and type:
java -version
// Should display: java version "17.0.x" (or your version)

javac -version
// Should display: javac 17.0.x

Step 3: Install an IDE

  • NetBeans: Beginner-friendly, good for learning
  • Eclipse: Popular, many plugins available
  • IntelliJ IDEA: Professional-grade, excellent code assistance
  • VS Code: Lightweight, with Java Extension Pack

Step 4: Compile and Run from Command Line

// Save your code as HelloWorld.java

// Compile the program (creates HelloWorld.class)
javac HelloWorld.java

// Run the program
java HelloWorld

// Output: Hello, World!
